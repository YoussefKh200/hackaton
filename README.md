# Women in Engineering & Cybersecurity Hackathon Project

## Overview
This repository contains the project developed during the **Women in Engineering & Cybersecurity Hackathon**, held on **July 6, 2025**. The project was awarded **First Place**, recognizing its innovation, technical execution, and alignment with the hackathonâ€™s mission: empowering women in engineering and advancing cybersecurity solutions.

## Purpose
The project was designed to:
- Promote inclusivity and representation of women in engineering fields.
- Address pressing challenges in cybersecurity through practical, scalable solutions.
- Demonstrate how diverse perspectives can strengthen technical innovation.

## Features
- **Secure Architecture**: Implements best practices in data protection and system resilience.
- **User-Centric Design**: Focuses on accessibility and usability for diverse audiences.
- **Educational Component**: Provides resources and interactive modules to raise awareness about cybersecurity risks and mitigation strategies.
- **Scalable Implementation**: Built with modular components to allow future expansion and integration.

## Technology Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP (MVC architecture)
- **Database**: MySQL (normalized schema with secure queries)
- **Tools**: PHPMyAdmin, Browser DevTools
